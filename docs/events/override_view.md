# override_view

## Description

Lets you override the camera position and angles.

## Arguments

| Index | Name | Type |
| --- | --- | --- |
| 0 | Argument #0 | table |

### #0 - Properties

| Name | Description |
| --- | --- |
| x | Camera X position |
| y | Camera Y position |
| z | Camera Z position |
| pitch | Pitch view angle |
| yaw | Yaw view angle |
| fov | Field of view |
