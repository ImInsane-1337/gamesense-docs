# client.set_clan_tag

Sets the player's clan tag. The clan tag is removed if no argument is passed or if it is an empty string. Additional arguments will be concatenated.

```typescript
client.set_clan_tag(...)
```

## Arguments

| Name | Description | Type |
| --- | --- | --- |
| clan_tag | N/A | string |
