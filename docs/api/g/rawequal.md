# rawequal

Checks whether `v1` is equal to `v2`, without invoking any metamethod.

```typescript
rawequal(v1: any, v2: any): boolean
```

## Arguments

| Name | Description | Type |
| --- | --- | --- |
| v1 | The first value to compare. | any |
| v2 | The second value to compare. | any |

Returns `boolean`
