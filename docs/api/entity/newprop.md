# entity.new_prop

Creates a new entity prop that can then be read and modified with [entity.get_prop](newprop.md) / [entity.set_prop](newprop.md).

```typescript
entity.new_prop(propname: string, offset: number, type: number, array_type: number, array_element_size: number, array_count: number)
```

## Arguments

| Name | Description | Type |
| --- | --- | --- |
| propname | N/A | string |
| offset | N/A | number |
| type | N/A | number |
| array_type | N/A | number |
| array_element_size | N/A | number |
| array_count | N/A | number |
