# Code Snippets

                                        
### Overview

                                        
These code snippets can help you implement certain features without reinventing the wheel and potentially making your code more complex.

                                        
### How to submit a snippet

                                        
Open a pull request on [GitHub](https://github.com/gamesensical/docs) with your snippets in markdown-formatted files. Use the existing set of snippets as examples
                                            on how to format your markdown files, and how to name them.
