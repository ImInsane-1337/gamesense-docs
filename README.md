# GameSense Lua API Docs

Документация по API чита GameSense для создания Lua-скриптов в CS:GO.

## Структура каталога

* docs/api.md — основной справочник функций API и событий.
* docs/snippets.md — готовые примеры кода для различных задач.
* CONVENTIONS.md — стандарты кодирования, руководство по PUI и системный промпт (правила написания кода, особенности PUI, рендеринга и событий) для ИИ-ассистентов с протоколом уточнения требований.

## Поддерживаемые модули

Документация включает описание встроенных библиотек чита:
* client — общие функции игры и взаимодействия с читом.
* entity — работа с сущностями, игроками и NetProps.
* ui — создание стандартного интерфейса и меню.
* renderer — функции отрисовки геометрических примитивов, текста и текстур в paint-колбэках.
* database — персистентное сохранение настроек и данных на диск.
* cvar — чтение и запись консольных переменных игры.
* materialsystem — управление материалами и чамсами.
* panorama — интеграция и выполнение JS в Panorama UI игры.
* plist — индивидуальные настройки для списка игроков.
* config — экспорт и импорт настроек чита.

## Разработка интерфейса (PUI)

Для разработки пользовательского интерфейса рекомендуется использовать библиотеку PUI (Perfect UI), подключение которой осуществляется через require("gamesense/pui"). Подробные правила использования PUI и сравнение со стандартным ui.* описаны в файле CONVENTIONS.md.
